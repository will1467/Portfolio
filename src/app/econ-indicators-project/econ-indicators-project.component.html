GDP Live

An Academic project developed as part of the Massey INMS Scholarship Program, with the goal of estimating key economic indicators such as GDP using machine learning algorithms, broken down by region and industry. The project involved the creation of a website running on a NodeJS server with the capability to display real-time economic data responsively in a dashboard format. This app has now gone live under the name GDP Live


Technologies Used

ReactJS, Plot.ly, NodeJS, Python
