Chirper - Bird Song Analysis

An Academic Project for Massey University, the goal of the web application "Chirper" was to act as a front-end interface for a server which would analyse bird song using machine learning algorithms. The website allows users to upload audio files, visually analyse them using spectrograms and then send chunks of the bird song - known as "syllables" to the back-end server


Technologies Used

Django Web Server, Python, MySQL
